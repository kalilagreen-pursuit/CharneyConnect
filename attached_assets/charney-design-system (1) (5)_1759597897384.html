Brand Principles

Typographically Driven

The identity relies solely on HEX Franklin (Franklin Gothic) with no supporting typefaces. This creates a distinctly American, bold, confident voice with deliberate intentionality.

Simplicity & Strength

Every element should communicate both creative spirit and robust expertise. Avoid clutter. Let typography and color create visual hierarchy and impact.

Color Contrast Rule

When red (#FF5959) is used as a background, text must be black (#000000). This ensures readability and maintains the bold, graphic nature of the brand.

Mixed Color Headlines

The signature brand treatment combines black and red text in headlines to create visual interest and emphasize key words or phrases.

Design Principles for Developers

Do's

  • Use uppercase typography for headers and CTAs to create impact
  • Mix black and red in headlines for the signature Charney treatment
  • CRITICAL: When red is a background, text MUST be black (#000000)
  • Let typography create hierarchy - rely on scale and weight
  • Use Charney Red sparingly for maximum visual impact
  • Embrace white space - simplicity is strength
  • Use 6px red accent bars as section dividers
  • Stick to the spacing system for consistency

Don'ts

  • Don't use additional typefaces - Franklin Gothic only
  • Don't use colored text on red backgrounds - always use black
  • Don't over-use red - maintain its power as an accent
  • Don't add unnecessary decorative elements
  • Don't use gradients or complex effects
  • Don't use lowercase for major headlines
  • Don't deviate from the defined color palette
